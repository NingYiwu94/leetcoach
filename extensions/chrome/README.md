# LeetCoach Chrome 同步组件

当前版本：`1.2.0`

## 安装 / 更新

1. 在 Chrome 地址栏打开 `chrome://extensions/`。
2. 打开右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录：`leetcoach/chrome_extension`。
5. 如果以前安装过，请点击扩展卡片上的“刷新/重新加载”，并确认版本为 `1.2.0`。

## 新同步方式：Localhost Push

LeetCoach 打开后会在本机启动一个小服务：

```text
http://127.0.0.1:8765
```

扩展在力扣页面中读取到最近提交记录后，会自动推送到：

```text
http://127.0.0.1:8765/leetcode-submissions
```

这意味着用户正常刷题即可，不需要每做完一道题手动导出。

## 隐私说明

- 不读取账号密码。
- 不保存 Cookie。
- 不绕过力扣反爬。
- 只把浏览器页面中已经能看到的最近提交记录推送到本机 LeetCoach。
- 推送地址只使用 `127.0.0.1`，不会发送到公网服务器。

旧的 `127.0.0.1:18777` 主动桥接方式仍保留为备用。
